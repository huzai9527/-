## 环境安装
- 你可以按照下面的方式下载代码到本地
 ```bash
 git clone --depth 1 https://github.com/huzai9527/studyAnalysis.git SUSmember-dev
 ```
- 本项目在**python 3** 下运行,并且依赖**pands**,你可以在终端运行以下命令进行安装
```bash
pip install pandas
```
